# DVWA-WebGoat-
This repository contains my university security project developed as part of a “Special Topics” course focused on fundamental concepts of network and web security
During the practical labs, I deployed Damn Vulnerable Web Application (DVWA) using XAMPP inside a virtual machine environment and explored common web vulnerabilities in a controlled setup. This hands-on practice helped me understand how real-world attacks are performed and how security weaknesses can be exploited.

For the final project, I deployed OWASP WebGoat using Docker and performed vulnerability discovery and analysis using Burp Suite. The project focuses on identifying and analyzing common web attacks such as SQL Injection (including Blind Injection), Cross-Site Scripting (XSS), and Cross-Site Request Forgery (CSRF).

A detailed PDF report documenting the attack process, findings, and security analysis is included in this repository.

This project strengthened my practical understanding of web application security, vulnerability assessment, and penetration testing fundamentals
